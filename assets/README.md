# Repository Assets (assets/)

본 디렉토리는 기계학습 모델 아키텍처 및 파이프라인의 설계 도식화를 위한 그래픽 리소스를 보존하는 공간입니다.

## 1. 하위 폴더 구성

* **`diagram_sources/`**: 보고서 및 설명서에 탑재된 구조도 다이어그램의 편집용 원천 벡터 SVG 파일들을 보관하는 디렉토리입니다.

## 2. 용도 및 보존 사유

* 본 디렉토리에 저장된 이미지들은 모델의 전처리 필터링, R-stage 피처 확장 흐름도 등의 학술 도식화 자료입니다.
* GitHub Pages 배포나 대외 공유 시 그림 품질 저하 없이 영구 편집 가능하도록 원천 SVG 포맷으로 이관 보존합니다.
