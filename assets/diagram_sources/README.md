# Diagram Vector Sources (assets/diagram_sources/)

본 디렉토리는 학령인구 분석 모형의 학술 다이어그램 원천 파일인 SVG(Scalable Vector Graphics) 리소스를 보관합니다.

## 1. 다이어그램 소스 명세

* **`source_01_event_school_separation.svg`**: 신설, 통폐합 등 예외 변동을 유발하는 이벤트성 학교(2,173교)와 예측 대상 일반 학교(9,722교)의 정량 정제 필터링 흐름도
* **`source_02_school_isolation.svg`**: 학교알리미 위경도 및 구면 Haversine을 활용하여 학교 간 최단 거리 및 인접 영역을 탐색하는 공간 고립도 지수 산출 개념도
* **`source_03_r4_segment_split.svg`**: 세그먼트별 오차 진단 및 분석 검증 영역 분리도
* **`source_04_r5_actual_cohort.svg`**: 7년 전 지역 출생아 코호트 인구 통계를 초등 입학생수 피처로 매핑하는 코호트 모델 개념도
* **`source_05_training_pipeline.svg`**: 20종의 알고리즘 교차 검증 및 최적화 다출력 HistGradientBoosting Regressor 피처 적재 파이프라인 흐름도
