# Project Deliverables (deliverables/)

본 디렉토리는 기계학습 모델 개발 및 분석의 최종 결과물로 제출되는 학술/행정 산출물을 보관하는 공간입니다. 본 폴더는 실행 가능한 소스코드가 아닌, 대외 제출용 완성본 문서들을 분리 격리하여 보존합니다.

## 1. 하위 폴더 구성

* **`report/`**: 한글(HWP) 최종 보고서 작성용 원고 폴더
  * `FINAL_HWP_INSERTION_REPORT_V8.md`: 한글 보고서 서식에 즉시 복사하여 쓸 수 있도록 마크다운 형식으로 편집된 V8 보고서 본문 원고
  * `FIGURE_TABLE_ORDER_V8.md`: 보고서 본문에 삽입되는 그림/표의 순서 및 캡션 일치화 가이드
  * `images/v1/`: 보고서에 직접 삽입해야 하는 실제 분석 그래프 및 시나리오 대시보드 스크린샷 이미지 12종
  * `supporting_docs/`: R-stage 피처 명세서, 성능 오차 평가지표 가이드 등 보고서 작성을 보조하는 상세 문서셋
* **`paper/`**: 학술 대회 논문용 HTML/PDF 패키지 폴더
  * `paper_html_v5/`: A4 2페이지 분량을 가득 채우도록 3개 핵심 문단(웹 활용 시나리오, 공간 고립 한계, 실무 유의점)이 최종 보강된 다단형 HTML 원고 파일 및 스타일시트 패키지

---

## 2. 활용 안내

* **한글 보고서 본문 작성**: `report/FINAL_HWP_INSERTION_REPORT_V8.md` 텍스트를 전체 선택하여 한글 서식에 복사한 뒤, `FIGURE_TABLE_ORDER_V8.md` 순서에 맞춰 `report/images/v1/` 내의 그래프들을 한글 **[그림 넣기] (Ctrl + N, I)** 기능으로 바인딩합니다.
* **학술 논문 2페이지 PDF 변환**: `paper/paper_html_v5/index.html` 파일을 Chrome/Edge 브라우저에서 실행하고 **`Ctrl + P`** 인쇄 기능(용지 A4, 배경 그래픽 체크 활성화, 배율 90~100% 최적화)을 이용해 PDF 파일로 안전하게 익스포트하여 보관하십시오.
