# 학술 논문 형식 HTML 원고 템플릿 v5 (최종 2페이지 분량 최적화)

본 패키지는 학술 논문 형식의 레이아웃을 제공하는 최종 2페이지 분량 완결형 HTML/CSS 템플릿입니다. 기존 `paper_html_v4` 패키지를 보완하여, A4 2페이지 인쇄/PDF 저장 시 발생하던 하단 빈 여백 문제를 극복하기 위해 본문 내에 학술적 깊이를 더해주는 3개 문단을 자연스럽게 보강 탑재하였습니다.

## 1. 파일 구성

* `index.html`: A4 2단 레이아웃 논문 원본 HTML 파일 (그림 3개 탑재)
* `paper.css`: 다단 구성 및 그림 높이 제한, 인쇄 미디어 쿼리가 포함된 스타일시트
* `assets/`: 논문에 사용되는 3개의 이미지 리소스가 보관된 폴더
  * `fig05_r_stage_structure.png` (그림 1)
  * `fig07_model_performance_horizon.png` (그림 2)
  * `web_02_map_detail_panel.png` (그림 3)
* `export_guide.md`: Chrome/Edge 브라우저에서의 PDF 저장 상세 가이드 (배율 90~100% 최적화)
* `quality_check.md`: 품질 점검 결과서

## 2. 주요 개선 및 보강 사항

* **본문 텍스트 확장 (하단 여백 최소화)**:
  * **4장 웹 대시보드 활용 시나리오 문단**: 웹 대시보드에서 연도별 추이 분석 후 고립 학교를 우선 식별하는 행정 담당자의 실제 가상 활용 시나리오를 추가했습니다.
  * **5장 한계 및 향후 연구 문단**: 구면직선거리 중심 고립도 분석의 한계를 고백하고, 대중교통 노선 및 정밀 GIS 통학 시간 연계 등 미래 개선 방향을 보강했습니다.
  * **5장 실무 적용 시 유의점 문단 (선택 문단)**: 본 대시보드가 행정 판단의 보조 분석 도구이며, 학교 구성원 및 주민 의견과 통학 안전성 검토가 수반되어야 함을 명확히 명시했습니다.
* **최종 그림 3개 유지**:
  1. `(그림 1) R0~R5 R-stage 피처 확장 구조`
  2. `(그림 2) 예측 기간별 최종 HistGB 모델 성능 변화`
  3. `(그림 3) 학교 단위 지도 및 상세 진단 화면` (대표 지도 화면 `web_02_map_detail_panel.png` 1장 유지)
* **레이아웃 안정성 유지**: `column-span`과 `figure-wide` 속성을 완전히 제거하여 2단 다단 정합성이 인쇄 설정에서 자연스럽게 유지됩니다.

## 3. 실행 및 PDF 변환 방법

> [!IMPORTANT]
> * **브라우저 뷰 확인**:
>   일반 웹 브라우저 화면에서는 HTML 문서 특성상 하나의 긴 페이지 형태로 연속되어 보입니다. 실제 A4 규격의 페이지 분할 상태 및 레이아웃은 Chrome/Edge 브라우저의 **인쇄 미리보기(Ctrl + P)** 화면 또는 PDF 저장 기능을 통해 확인해야 합니다.
> * **PDF 변환 권장 옵션**:
>   - 브라우저에서 `index.html`을 열고 `Ctrl + P`를 입력합니다.
>   - 대상을 **PDF로 저장**으로 선택하고 용지 크기를 **A4**로 지정합니다.
>   - **배경 그래픽** 옵션을 반드시 체크합니다.
>   - v5 본문은 텍스트량이 보강되었으므로 배율 옵션을 **`90% ~ 100%`** 범위로 설정하여 2페이지가 꽉 차도록 내보낼 수 있습니다.
>   - 자세한 출력 팁은 `export_guide.md` 파일에서 확인하실 수 있습니다.
