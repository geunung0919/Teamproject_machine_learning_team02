# 그림 및 표 한글(HWP) 삽입 순서표 v8 (FIGURE_TABLE_ORDER_V8.md)

본 문서는 최종 결과보고서 한글(HWP) 파일 작성자가 혼선 없이 시각화 이미지(PNG)와 Markdown 표 데이터를 삽입할 수 있도록, 순서대로 삽입 위치, 제목 규칙, 파일 경로 및 활용 목적을 정리한 통합 가이드맵입니다.

---

### 1. 편집 서식 기본 원칙 (필수)
* **표 제목**: 반드시 표 **위(Top)**에 작성합니다. (예: `<표 1> 데이터 수집 및 원천 구성표`)
* **그림 제목**: 반드시 그림 **아래(Bottom)**에 작성합니다. (예: `<그림 1> 2018~2025년 학령인구 및 학생 수 변화 추이`)
* **이미지 해상도**: 생성된 모든 PNG 자원은 해상도 150 DPI 이상으로 출력되어 고화질 삽입에 최적화되어 있습니다. 한글 편집기에서 마우스 오른쪽 버튼 클릭 후 `마우스로 크기 지정`을 해제하고 원본 크기 대비 비율을 조절하여 배치하는 것을 권장합니다.
* **편집자 주의**: `FINAL_HWP_INSERTION_REPORT_V8.md` 문서에는 한글 복사/붙여넣기를 편하게 하도록 HTML table 관련 태그가 완전히 배제되고 표 제목이 일반 텍스트로 정리되어 있습니다. 따라서 그림 삽입은 본 순서표의 경로 및 위치 가이드를 참고하여 직접 진행하시기 바랍니다.
