# Evaluation Metrics and Model Performance (docs/evaluation_metrics.md)

본 문서는 학생수 감소압력 예측 모델의 성능을 정밀 다각화 진단하기 위해 채택한 6대 평가지표의 정의와 Horizon별 최종 검증 수치 결과를 기술합니다.

---

## 1. 6대 평가 모델 진단 지표 정의

* **`MAE (Mean Absolute Error)`**: 실제 학생수와 예측 학생수 간 차이의 절대 평균으로, 보편적인 수치적 오차의 크기를 의미합니다.
* **`RMSE (Root Mean Squared Error)`**: 오차 제곱 평균의 제곱근으로, 큰 예측 오차가 발생한 이상치 학교군에 더 강한 벌점을 부과하여 모델 불안정성을 체크합니다.
* **`Level R² (결정계수)`**: 학생 수의 절대 규모 레벨을 모델이 얼마나 잘 설명하는지 나타내는 지표입니다.
* **`Delta R² (증감 설명력 결정계수)`**: 절대 규모가 아닌, 연도별 학생 수의 **'순 증감 변화량'**에 대한 모델의 추세 추적력을 평가하는 핵심 지표입니다. (1에 가까울수록 감소 추세 예측력이 뛰어남)
* **`Median AE (중앙값 절대오차)`**: 오차 분포의 중앙값으로, 특정 특이치 학교(예: 대규모 신도시 아파트 입주로 유입이 비정상 폭발한 학교)에 의한 평균 왜곡을 배제하고 일반적인 학교의 오차 수준을 보여줍니다.
* **`저학생수 학교 MAE`**: 농산어촌 등 전교생 규모가 작은 소규모 취약 학교군에 한정한 MAE로, 지역 교육공백 지수 산출 시의 정밀 신뢰도를 입증합니다.

## 2. 최종 모델 검증 성능 요약

### 나. 1~5개년 종합 평균 성능
* **1~5년 평균 MAE**: `34.380397명`
* **1~5년 평균 Delta R²**: `0.443568`

저학생수 학교군의 5년 장기 MAE 오차가 **`9.244명`**으로 전체 평균 대비 극히 미미하게 통제되어, 향후 지방 소규모 학교의 교육공백 리스크를 모니터링하기 위한 행정 보조 지표로써 충분한 과학적 신뢰성을 획득하였습니다.
