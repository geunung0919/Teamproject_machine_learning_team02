# 최종 모델 기준 점검

## 정리 결과

정책 후보 재현 모델은 실제 검증 목적과 맞지 않아 제거했다.  
현재 발표 기준 분류 성능은 `2009~2018 학습 / 2019~2022 검증`의 EDSS 다음 해 소멸 proxy 모델만 사용한다.

## 회귀

| 구분 | 모델 | 성능 기준 |
|---|---|---|
| 베이스 | Ridge | 시군구 다음 해 학령인구 예측 |
| 튜닝 | RandomForest | 시군구 다음 해 학령인구 예측 |

| 모델 | MAE | RMSE | R2 |
|---|---:|---:|---:|
| Ridge | 642.4 | 943.9 | 0.9993 |
| RandomForest | 566.0 | 1004.6 | 0.9992 |

## 분류

| 구분 | 모델 | 성능 기준 |
|---|---|---|
| 베이스 | Logistic Regression | 2019~2022 EDSS 다음 해 소멸 검증 |
| 튜닝 | HistGradientBoosting | 2019~2022 EDSS 다음 해 소멸 검증 |

| 모델 | Precision | Recall | F1 | ROC-AUC | PR-AUC |
|---|---:|---:|---:|---:|---:|
| Logistic | 0.695 | 0.717 | 0.706 | 0.970 | 0.608 |
| HistGB | 0.688 | 0.717 | 0.702 | 0.983 | 0.725 |

분류모델은 학습-추론 불일치를 줄이기 위해 서빙 호환성이 낮은 인구이동 원자료/상권 원자료/파생비율 피처를 제외하고 10개 피처로 재학습했다.
F1은 Logistic이 소폭 앞서지만, 튜닝한 HistGB는 ROC-AUC/PR-AUC가 더 높아 위험 후보 랭킹 성능이 좋다.

## 폐기한 항목

아래 항목은 `archive/policy_reproduction_experiment_20260518`로 이동했다.

- `train_consolidation_candidate_classifier.py`
- `consolidation_candidate_classifier_metrics.csv`
- 정책 후보 label 재현 관련 예측/리포트

폐기 이유:

- 정답 라벨이 실제 폐교/소멸이 아니라 우리가 만든 정책 후보 label이었다.
- 따라서 F1 0.996은 실제 검증 성능이 아니라 규칙 재현 성능이었다.
- 발표용 성능지표로 쓰면 오해가 생기므로 제외했다.

## 남은 한계

2019~2022 검증 모델도 실제 폐교확정 명부가 아니라 EDSS 학교ID 소멸 proxy를 사용한다.  
또한 과거 EDSS에는 좌표가 없어 학교 고립도, 대체학교 거리, 학교 반경 상권 피처를 직접 넣을 수 없다.

따라서 발표 표현은 다음처럼 제한한다.

> 분류 성능은 EDSS 장기 패널에서 만들 수 있는 다음 해 학교ID 소멸 proxy 기준 검증이며, 최종 지도에서는 이 검증 결과와 별도로 현재 좌표 기반 고립도/상권/대체학교 피처를 정책 위험점수에 반영했다.
