# Web Data Export Specification (docs/web_data_export.md)

본 문서는 기계학습 추론 엔진 및 시나리오 분석 모듈의 연산 결과를 웹 대시보드 상에 시각화하기 위해 `public/data/` 경로로 최종 변환 및 내보내기(Export)된 정적 데이터 명세를 정의합니다.

---

## 1. public/data/ 내보내기 파일 구조

웹 대시보드의 원활한 기동을 위해 데이터는 다음과 같이 정형화된 JSON 및 CSV로 구조화되어 저장됩니다.

* **`final_scenario_school_web.json`**:
  * **대상**: 안정 예측 대상 9,722개교의 2026~2030년 미래 예측치 및 고립 지표 정보
  * **필드 구성**:
    * `school_code`: 학교 고유 코드
    * `school_name`: 학교명
    * `lat` / `lng`: 위·경도 좌표
    * `predictions`: 2026~2030년 연도별 학생 수 추이 리스트
    * `isolation_score`: 공간 고립도 점수
    * `is_priority_review`: 감소압력과 고립이 결합된 **우선점검 지원 여부 플래그** (`0` 또는 `1`)
* **`excluded_school_web.json`**:
  * **대상**: 시계열 모델 훈련 노이즈 방지를 위해 분격 격리되었던 2,173개교의 정보
  * **용도**: 웹 대시보드 상에서 누락 없이 전체 학교 마커를 렌더링하고, 2025년 기준 상수를 유지하며 지도에 표현
* **`summary_national_by_year.csv`**:
  * **대상**: 2025~2030년 전국 합계 학생 수 변화, 연도별 누적 감소율, 시도별 감소폭 등을 요약한 데이터
  * **용도**: 대시보드 메인 화면의 대형 트렌드 차트 및 카드 컴포넌트 렌더링용

## 2. 웹 대시보드 UI 연동 의미

* **학교별 지도 마커 (Map Marker)**:
  * 지도 상의 학교별 점의 색상은 감소압력의 속도 및 고립 여부에 따라 변동합니다. (우선점검 지원 필요 시 시각적 경고 신호로 반짝임)
* **연도·지역·학교급 필터**:
  * 행정 담당자가 슬라이더 및 드롭다운을 조작하여 2026~2030년 중 목표 연도의 위험 분포와 특정 학교급(초등/중등/고등)의 고립 분포를 직관적으로 필터링하도록 설계되었습니다.
* **상세 진단 패널 (Detail Panel)**:
  * 마커 선택 시 우측 상세 패널에 해당 학교의 예측 학생 수 하강 그래프, 감소율, 최단 거리 동일급 학교 분포, 고립도 점수 등이 실시간으로 집계 렌더링됩니다.

---

## 3. 실무 정책적 사용 시의 한계 명시

> [!IMPORTANT]
> * **의사결정 보조 도구의 정의**:
>   - 본 내보내기 데이터와 대시보드 상에 표시되는 모든 '우선점검', '우려' 플래그는 통폐합이나 폐교를 기계적으로 자동 확정하는 행정적 판정 기준이 아닙니다.
>   - 본 시스템은 인구 하락과 지리 공간 소멸 위기에 놓인 취약 지역 학교를 선별하여 행정 검토 대상을 압축해 주는 **우선점검 참고 및 의사결정 보조 분석 도구**로 기능합니다.
>   - 최종 규모 조정 여부는 각 지역의 정밀한 통학 안전성, 주민 소통 결과 및 시도교육청 고유의 절차를 통해 독자 판단되어야 합니다.
