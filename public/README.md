# Web Dashboard Static Data Package (public/)

본 디렉토리는 학령인구 감소압력 및 공간적 고립 리스크 지수를 지리 정보(Map) 및 상세 대시보드로 분석해 주는 웹 대시보드 애플리케이션의 **정적 데이터 패키지** 보관소입니다.

> [!IMPORTANT]
> * **단독 실행 한계 명시**:
>   - 본 폴더는 프론트엔드 UI를 렌더링하는 실행 파일(HTML/JS/CSS)을 담지 않고, 시나리오 분석 엔진에서 내보낸 최종 예측 CSV 및 JSON 파일만을 탑재하고 있습니다.
>   - 따라서 **이 폴더 단독으로는 웹 UI 대시보드가 브라우저 상에 실행되지 않습니다.**
>   - 웹 대시보드 프론트엔드 서비스 소스코드는 별도의 웹 배포 저장소 또는 다른 모듈에서 관리되며, 본 폴더 내의 정적 데이터를 수집하여 렌더링하도록 연동되어 있습니다.

---

## 1. 정적 데이터 파일 스펙 (`public/data/scenario_v5_v2/`)

웹 UI 시각화를 위해 기동되는 데이터 파일 구성은 다음과 같습니다.

* `final_scenario_school_web.json` (23MB): 2026~2030 미래 예측 5개년 수치, 구면 haversine 공간 고립 점수, 우선점검 검토 대상 플래그를 담은 핵심 지도 렌더링용 JSON
* `excluded_school_web.json` (3.1MB): 시계열 모델 왜곡을 방지하기 위해 분리 정제되었던 이벤트성 학교군의 기본 정보 및 좌표 JSON
* `summary_national_by_year.csv`: 전국 기준 연도별 예측 총합계 추이 요약
* `summary_sido_by_year.csv`: 17개 시도 단위의 2025~2030 연도별 변화 및 누적 감소 통계
* `top_priority_national_2030.csv`: 2030년 시점 기준 감소압력 및 고립 리스크가 결합되어 가장 최우선적으로 통학 안전이나 행정 검토가 필요한 상위 소규모 학교 리스트
