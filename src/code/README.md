# V5 Core Model Stages, Scenarios, and Web Exports

This folder contains the core modeling and inference scripts for the V5 student count pipeline.

## Sub-modules

- `r_stages/`: Folders containing `run_stage.py` and `README.md` for each R-stage experiment (R0 to R6) as well as the final model selection logic.
- `scenario/`: Scripts to generate future student predictions, export packaged data for visual dashboards, and layer excluded event schools.
