# R5 - Cohort Proxy

- **Purpose**: Evaluates regional and national aggregate cohort trend proxies.
- **Input**: Model views combined with aggregate trend CSV files.
- **Output**: Multi-horizon predictions and metrics comparison.
- **Feature Group**: All R3 features + national, regional, and municipal student totals, growth slopes, and school student shares.
- **Model Group**: LinearRegression, Ridge, RandomForest, HistGradientBoosting.
- **Relationship to Other Stages**: Intermediate stage prior to actual cohort demographics integration.
