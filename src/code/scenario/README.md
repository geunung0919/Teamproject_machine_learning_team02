# Scenario Generation and Web Exports

This module coordinates scenario generation, layers anomalous schools, and packages final dashboard assets.

## Modules

- `generate_final_scenario.py`: Predicts future school student counts.
- `add_excluded_event_layer.py`: Preserves anomalous historical schools in summary aggregates.
- `export_web_package.py`: Creates clean, dashboard-ready JSON/CSV targets.
