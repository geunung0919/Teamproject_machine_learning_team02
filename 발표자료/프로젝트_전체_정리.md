# 프로젝트 전체 정리

## 1. 프로젝트 주제

주제는 `학령인구 예측 및 통폐합 위험군 학교 분류`이다. 핵심 목표는 전국 학교를 대상으로 2026년부터 2040년까지 학생 수 변화를 예측하고, 학령인구 감소와 지역 여건을 함께 고려해 통폐합 검토 우선순위를 시각화하는 것이다.

프로젝트는 두 축으로 구성된다.

```text
1. 학령인구/학생수 예측: 회귀 및 시나리오 모델
2. 통폐합 위험군 산정: supervised 보조모델 + 정책 위험점수 산식
```

## 2. 프로젝트 차별점

기존 유사 연구는 주로 출산율 감소, 폐교 가능성, 폐교 후 통학거리 증가에 집중한다. 이 프로젝트는 여기에 다음 피처를 추가해 정책 해석력을 높였다.

```text
학교 고립도:
가장 가까운 같은 학교급 거리, 5km 내 같은 학교급 수, 대체학교 접근성

상권 취약도:
학교 주변 500m/1km/2km 상권 수, 교육/아동/의료 관련 업종 밀도

지역 감소위험:
시군구 인구, 출생, 출산율, 인구이동, 학령인구 구조

EDSS 객관 위험:
과거 EDSS 학교ID 소멸 proxy 기반 폐교 패턴 유사도
```

따라서 최종 주장은 “정확한 폐교 예언”이 아니라 “학령인구 감소 시대에 어느 학교를 우선 검토해야 하는지 보여주는 의사결정 보조 모델”이다.

## 3. 사용 데이터

현재 사용한 주요 데이터는 다음과 같다.

| 구분 | 데이터 | 활용 |
|---|---|---|
| 학교 현황 | 교육재정알리미/학교 현황 | 현재 학교 목록, 학교급, 좌표, 학생수 |
| 과거 학교 패널 | EDSS 2009~2023 | 폐교 proxy 라벨, 객관 위험 보조모델 |
| 인구 | KOSIS 시군구 인구 | 학령인구 예측 |
| 출생 | KOSIS 출생아수 | 출생 코호트 시나리오 |
| 출산율 | KOSIS 합계출산율 | 장기 감소 경로 해석 |
| 인구이동 | KOSIS 전입/전출 | 지역 감소위험 |
| 상권 | 소상공인시장진흥공단 상가업소 | 학교 주변 생활 기반/상권 취약도 |

## 4. 전체 파이프라인

최종 파이프라인은 다음 순서로 이해하면 된다.

```text
1. 전국 원본 데이터 수집
2. KOSIS 인구/출생/출산율/이동 데이터 전처리
3. 소상공인 상권 데이터 시군구/학교 반경 피처 생성
4. EDSS 과거 학교 패널 기반 폐교 proxy 학습 데이터 생성
5. 현재 학교별 피처 생성
6. 시군구 학령인구 회귀모델 학습
7. 출생 코호트 기반 장기 시나리오 생성
8. 학교별 2026~2040 학생수 시나리오 생성
9. supervised 보조 폐교 패턴 모델 학습
10. 정책 위험점수 산식으로 최종 위험등급 산출
11. 모델 비교/피처 중요도/품질 리포트 생성
12. Folium/Leaflet 기반 대화식 HTML 지도 생성
```

최종 실행 runner:

```text
src/run_final_pipeline.py
```

## 5. 회귀모델 설계

회귀모델은 시군구 단위 학령인구를 예측하고, 그 압력비를 학교별 학생수에 적용하는 구조이다.

사용 피처 예시:

```text
현재 시군구 학령인구
연령대별 인구 구조
출생아수
합계출산율
전입/전출/순이동
상권 요약 피처
전년도 변화량
```

비교 모델:

```text
baseline_previous_year_population:
전년도 학령인구를 그대로 다음 해 예측값으로 사용

ridge:
선형 baseline 성격의 회귀모델

random_forest:
최종 후보 회귀모델
```

최신 성능:

| 모델 | MAE | RMSE | R2 |
|---|---:|---:|---:|
| 전년도 유지 baseline | 1142.6 | 1667.6 | 0.9979 |
| Ridge | 642.4 | 943.9 | 0.9993 |
| RandomForest | 566.0 | 1004.6 | 0.9992 |

해석상 주의:

- R2가 높은 이유는 시군구별 인구 규모 차이가 크기 때문이다.
- 발표 비교는 Ridge를 회귀 베이스모델, RandomForest를 튜닝모델로 둔다.
- RandomForest는 Ridge 대비 MAE가 642.4명에서 566.0명으로 약 11.9% 낮다.
- 다만 RMSE는 Ridge가 더 낮으므로, “튜닝 모델이 모든 지표에서 압도적”이라고 말하지 않고 MAE 기준 개선과 비선형 패턴 반영을 강조한다.
- 출산율은 장기 원인 변수이고, 현재 학령인구는 그 결과가 누적된 직접 변수이다.

## 6. 출생 코호트 시나리오

기존 회귀 압력비 모델은 장기 감소를 약하게 반영하는 문제가 있어 출생 코호트 기반 시나리오를 추가했다.

핵심 아이디어:

```text
2007~2009년생이 빠져나가고,
2023~2025년생 및 이후 출생 코호트가 학교급별로 진입한다고 가정한다.
```

시나리오:

```text
기준: 최근 출생 수준 유지
낙관: 최근 출생 수준 대비 +10%
비관: 최근 출생 수준 대비 -10%
```

최종 HTML에서는 다음 모델을 직접 선택할 수 있다.

```text
출생 코호트 기준
변화량 모델 (인구이동 반영)
기존 시군구 압력비 기준
```

## 6-1. 변화량 모델 통합

장기 예측에서 지역별 인구이동 차이를 더 명확히 보기 위해 변화량 회귀모델을 최종 파이프라인에 추가했다.

```text
기존 절대값 모델:
다음 해 시군구 학령인구 절대값을 직접 예측

변화량 모델:
다음 해 시군구 학령인구 증감량을 예측한 뒤 현재 인구에 더함
```

변화량 모델은 15년 반복 예측에서 발산할 수 있으므로 다음 안전장치를 적용했다.

```text
연간 최대 감소: 현재 학령인구의 -20%
연간 최대 증가: 현재 학령인구의 +10%
예측 인구 최소값: 1명
```

변화량 모델의 역할은 “최종 성능 1위 모델”이 아니라, 인구이동과 연령구조가 지역별 감소 속도에 어떻게 반영되는지 보여주는 비교 시나리오다.

2040 전국 학생수 합계 비교:

| 모델 | 2040 예측 학생수 | 2025 대비 변화율 |
|---|---:|---:|
| 절대값/압력비 모델 | 3,679,172명 | -27.2% |
| 변화량 모델 | 2,648,950명 | -47.6% |
| 출생 코호트 기준 | 2,898,445명 | -42.6% |

시도별 감소율 표준편차:

| 모델 | 시도별 감소율 표준편차 | 해석 |
|---|---:|---|
| 절대값/압력비 모델 | 10.2 | 지역 차별화 약함 |
| 변화량 모델 | 21.9 | 지역별 인구이동 차이를 가장 크게 반영 |
| 출생 코호트 기준 | 14.1 | 출생 코호트 기반 장기 감소 반영 |

## 7. 분류모델 설계

분류모델은 실제 통폐합 확정 라벨이 부족하기 때문에 EDSS 학교ID 소멸 proxy를 사용했다.

라벨:

```text
EDSS 학교ID가 5년 뒤 사라지면 positive
```

비교 모델:

```text
baseline_logistic_regression:
발표용 기본 분류모델

final_context_hist_gradient_boosting:
최종 supervised 보조모델
```

최신 성능:

| 모델 | ROC-AUC | PR-AUC | Precision | Recall | F1 |
|---|---:|---:|---:|---:|---:|
| LogisticRegression | 0.942 | 0.388 | 0.469 | 0.413 | 0.439 |
| HistGradientBoosting | 0.958 | 0.573 | 0.619 | 0.487 | 0.545 |

해석상 주의:

- 이 모델은 폐교 확률을 직접 예측하는 모델이 아니다.
- EDSS 과거 소멸 패턴과 유사한 학교를 찾는 객관 보조모델이다.
- 최종 위험등급은 supervised 모델 결과와 정책 위험점수를 결합한다.

## 8. 최종 위험등급 산정

최종 위험등급은 다음 피처들을 조합한 정책형 위험점수로 산출한다.

```text
예측 학생수 규모
장기 학령인구 감소
학교 고립도
주변 대체학교 접근성
상권 취약도
지역 감소위험
EDSS 폐교패턴 유사도
특수/각종학교 여부
학생수 데이터 품질
```

위험등급:

```text
저위험
중위험
고위험 검토
통폐합 가능 고위험
교육공백 우려 고위험
특수/각종학교 별도검토
학생수 데이터 확인필요
```

중요한 해석:

- 통폐합 가능 고위험은 “학생수 감소 + 대체학교 접근 가능”에 가깝다.
- 교육공백 우려 고위험은 “학생수 감소 + 고립 + 대체학교 부족”에 가깝다.
- 상권/고립도는 성능 향상만을 위한 피처라기보다 정책 해석과 차별점 강화를 위한 피처이다.

## 10. 발표에서 강조할 포인트

발표에서는 다음 순서가 가장 자연스럽다.

1. 학령인구 감소 문제 제기
2. 기존 연구와의 차이: 고립도/상권/지역 감소위험 추가
3. 데이터 수집 범위: 전국 단위, 2026~2040
4. 회귀모델: 학령인구/학생수 예측
5. 출생 코호트 시나리오: 장기 감소 압력 보완
6. 분류모델: EDSS proxy 기반 객관 위험 보조
7. 최종 위험등급: 정책 해석형 점수
8. 지도 시연: 모델 선택, 연도 선택, 위험 피처 체크박스
9. 한계: 실제 통폐합 라벨 부족, 좌표 품질, 가중치 민감도
10. 결론: 폐교 예언이 아니라 통폐합 검토 우선순위 산정 도구

## 11. 현재 결론

현재 프로젝트는 단순한 ML 모델 하나가 아니라, 데이터 수집, 예측 모델, 객관 보조 분류모델, 정책 위험점수, 지도 시각화를 연결한 의사결정 보조 시스템에 가깝다.

가장 중요한 표현은 다음과 같다.

```text
본 프로젝트는 폐교 여부를 단정하는 모델이 아니라,
학령인구 감소와 지역 여건을 함께 고려해
통폐합 검토 우선순위를 제시하는 분석 도구이다.
```
