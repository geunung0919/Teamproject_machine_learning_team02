# 최종 실행 가이드

이 문서는 현재 프로젝트에서 발표/제출용 최종 산출물을 다시 만드는 기준 실행 순서다. 과거 프로토타입 스크립트는 `archive/legacy_scripts_20260516` 아래로 분리했고, 최종 실행은 아래 파일 중심으로 본다.

## 핵심 실행 명령

이미 수집된 데이터를 기준으로 모델과 지도만 다시 만들 때:

```powershell
python src/run_final_pipeline.py
```

API 원본 수집부터 다시 시작할 때:

```powershell
python src/run_final_pipeline.py --with-collect
```

## 최종 파이프라인 순서

1. `src/model_edss_closure_risk_national.py`
   - EDSS 학교ID 5년 뒤 소멸 여부를 proxy 라벨로 한 객관 분류 benchmark 생성
2. `src/final_national_training_pipeline.py`
   - 전국 학교 현재 피처 생성
   - 학령인구 기반 2026~2040 학생수 회귀 예측
   - 정책형 통폐합 위험점수/위험등급 생성
3. `src/final_supervised_training.py`
   - EDSS supervised 보조 백분위를 최종 시나리오에 결합
   - 이 값은 참고 피처이며 최종 위험점수를 직접 덮어쓰지 않음
4. `src/extract_model_feature_importance.py`
   - 회귀/분류 피처 중요도 CSV와 PNG 생성
5. `src/build_school_radius_commercial_features.py`
   - 학교 반경 500m/1km/2km 내 전체/교육/아동/의료 상권 수 생성
6. `src/analyze_risk_threshold_sensitivity.py`
   - 고립도/상권/지역감소 임계값 변화에 따른 고위험 학교 수 민감도 분석
7. `src/build_school_level_cohort_scenario.py`
   - 학교급별 출생 코호트 기반 낙관/기준/비관 시나리오 생성
8. `src/fertility_pathway_analysis.py`
   - 출산율이 출생아 수와 학령인구로 이어지는 경로 분석 생성
9. `src/build_final_interactive_school_risk_map.py`
   - 최종 Folium HTML 지도 생성
   - 좌표 품질, 학생수 확인필요, 특수/각종학교 별도검토 리포트 생성

## 최종 주요 산출물

- 지도: `outputs/maps/final_national_interactive_school_risk_scenario.html`
- 최종 시나리오: `data/processed/final_national_school_scenario_2026_2040.csv`
- 회귀 성능: `outputs/reports/final_national_population_regression_metrics.csv`
- 분류 성능: `outputs/reports/final_supervised_closure_classifier_metrics.csv`
- 회귀 피처 중요도: `outputs/reports/regression_feature_importance.csv`
- 분류 피처 중요도: `outputs/reports/classification_context_feature_importance.csv`
- 출산율 경로 분석: `outputs/reports/fertility_pathway_model_metrics.csv`, `출산율_경로_분석.md`
- 반경 기반 상권 피처: `data/processed/school_radius_commercial_features.csv`
- 위험 임계값 민감도 분석: `outputs/reports/risk_threshold_sensitivity.csv`
- 학교급별 코호트 시나리오: `data/processed/school_level_cohort_scenario_2026_2040.csv`
- 코호트 시나리오 요약: `outputs/reports/school_level_cohort_scenario_total_summary.csv`
- 좌표 품질: `outputs/reports/final_coordinate_quality_report.csv`
- 좌표 오류 상세: `outputs/reports/final_invalid_coordinates_2026_2040.csv`
- 학생수 확인필요: `outputs/reports/final_student_count_data_check_2026_2040.csv`
- 특수/각종학교 별도검토: `outputs/reports/final_special_school_review_2026_2040.csv`

## 발표에서 구분할 점

- 회귀모델은 2026~2040년 학교별 예측 학생수를 만든다.
- 지도 아래 피처 체크박스는 회귀 예측값을 바꾸는 기능이 아니라, 위험점수 산식에서 어떤 피처를 반영할지 보는 정책 시뮬레이션이다.
- 상권/고립도/지역감소위험은 “모델 성능이 무조건 좋아졌다”보다 “통폐합 판단 시 교육공백과 지역 맥락을 함께 볼 수 있다”는 차별점으로 설명하는 것이 안전하다.
