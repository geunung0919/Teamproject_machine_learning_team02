# 프로젝트 코드 구조 가이드

이 문서는 전체 코드 구조를 빠르게 찾기 위한 목차 파일이다.  
자세한 설명은 폴더별 문서로 분리했다.

## 전체 실행

최종 파이프라인:

```powershell
python src/run_final_pipeline.py
```

API 수집까지 포함:

```powershell
python src/run_final_pipeline.py --with-collect
```

## 공통 파일

### `src/run_final_pipeline.py`

최종 실행 순서를 관리하는 runner다.

- `run_step(title, script)`: 단계별 스크립트를 실행한다.
- `main()`: `--with-collect` 옵션에 따라 API 수집 포함 여부를 결정한다.

### `src/project_config.py`

프로젝트 공통 설정 파일이다.

- 위험등급 라벨/색상
- 위험점수 가중치
- 학교급 정규화
- 좌표 유효성 검사
- 위험점수/위험등급 산식

### `src/config_loader.py`

`.env`에서 API 키를 읽는 유틸리티다.

## 발표용 핵심 표현

- 회귀모델은 학생수/학령인구 예측을 담당한다.
- 위험등급은 회귀 예측값과 정책 해석 피처를 결합한 통폐합 검토 우선순위다.
- 상권/고립도/학령수요 감소압력은 성능 향상보다 정책 해석력과 차별점 강화를 위한 피처다.
- EDSS는 실제 통폐합 라벨 부족을 보완하는 proxy 검증/보조지표다.
