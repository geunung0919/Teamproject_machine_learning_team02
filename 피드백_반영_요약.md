# 피드백 반영 요약

`프로젝트_분석_피드백.md`의 핵심 지적을 기준으로 현재 프로젝트에 반영한 내용과 남은 한계를 정리한다.

## 핵심 반영 사항

### 1. 회귀모델 baseline 추가

가장 중요한 지적은 현재 회귀모델이 `현재 학령인구`를 사용해 `다음 해 학령인구`를 예측하므로 성능이 과대평가될 수 있다는 점이었다.

이를 반영해 `final_national_population_regression_metrics.csv`에 `baseline_previous_year_population` 행을 추가했다.

이 baseline은 다음 의미다.

```text
내년 학령인구 = 올해 학령인구
```

## 추가 반영: HTML 시각화 통합

출생 코호트 기반 장기 시나리오를 최종 HTML 지도에도 연결했다.

반영 파일:

```text
src/viz/build_final_interactive_school_risk_map.py
outputs/maps/final_national_interactive_school_risk_scenario.html
```

그래프 탭의 `그래프 종류`에 다음 항목을 추가했다.

```text
출생 코호트 장기 시나리오 비교
```

이 그래프는 기존 학교별 압력비 예측선과 출생 코호트 기준/낙관/비관 시나리오를 같이 보여준다. 위험등급 체크박스는 정책 위험점수 시뮬레이션용이므로, 코호트 그래프에는 적용하지 않고 지역/학교급 필터만 적용한다.

### 2. 모델 표현 수정

분류모델은 실제 통폐합 확정 라벨이 아니라 EDSS 학교ID 5년 내 소멸 proxy를 사용한다. 따라서 최종 표현은 다음처럼 정리한다.

```text
폐교 확률 예측 X
통폐합 검토 우선순위 산정 O
```

지도에서도 EDSS 값은 `폐교 확률`이 아니라 `EDSS 폐교패턴 유사도 백분위`로 표시했다.

### 3. 차별 피처의 역할 정리

학교 고립도, 상권 취약도, 지역 감소위험은 현재 두 가지 방식으로 사용된다.

- 정책 위험점수 산식에 직접 반영
- 지도에서 피처 토글/점수 기여 chip으로 해석 가능하게 표시

단, EDSS supervised 분류 학습에는 학교별 고립도와 학교 반경 상권이 직접 들어간 것이 아니다. 따라서 발표에서는 “분류 성능 향상”보다 “정책 해석력 강화”로 설명해야 한다.

### 4. 장기 예측 프레이밍 정리

2040년 결과는 확정 예측이 아니라 현재 출생, 출산율, 인구이동, 학령인구 구조가 이어질 때의 시나리오로 설명한다.

특히 2007~2009년 출생 코호트와 2023~2025년 출생 코호트의 약 54만 명 차이는 학령인구 감소 압력을 설명하는 핵심 근거로 사용할 수 있다.

## 남은 중요 한계

### 회귀모델 구조

현재 모델은 시군구 0~19세 학령인구 압력비를 학교별 학생수에 적용한다. 학년별 코호트 이동을 직접 추적하지 않는다.

후속 개선은 학교급별 코호트 모델이다.

```text
초등학교: 최근 출생아/0~4세 인구
중학교: 10~12년 전 출생 코호트
고등학교: 15~17년 전 출생 코호트
```

KOSIS 출생 데이터 수집 범위를 `2007~2025년`으로 확장했고, 학교급별 출생 코호트 기반 보조 시나리오를 추가했다.

생성 산출물:

```text
data/processed/school_level_cohort_scenario_2026_2040.csv
outputs/reports/school_level_cohort_scenario_summary.csv
outputs/reports/school_level_cohort_scenario_total_summary.csv
```

시나리오:

```text
비관: 미래 출생 코호트 2023~2025 평균 대비 -10%
기준: 미래 출생 코호트 2023~2025 평균 유지
낙관: 미래 출생 코호트 2023~2025 평균 대비 +10%
```

기준 시나리오의 전국 학생수 변화는 다음과 같다.

```text
2029년: -14.9%
2035년: -37.8%
2040년: -43.5%
```

이 값은 기존 최종 지도에 쓰인 시군구 학령인구 압력비 모델보다 감소폭이 크다. 따라서 현재는 최종 모델을 바로 대체하기보다, “출생 코호트 구조를 반영한 보조 장기 시나리오”로 함께 제시하는 것이 안전하다.

### 상권 피처

기존에는 시군구 단위 상권 요약만 사용했다. 이 한계를 줄이기 위해 학교 좌표와 소상공인 좌표를 BallTree로 매칭해 반경 기반 상권 피처를 추가 생성했다.

생성 산출물:

```text
data/processed/school_radius_commercial_features.csv
outputs/reports/school_radius_commercial_feature_summary.csv
```

반경:

```text
500m, 1km, 2km
```

업종:

```text
전체 상권, 교육 업종, 아동/키즈 관련, 의료 업종
```

단, 아직 이 반경 상권 피처를 최종 위험 산식에 완전히 대체 적용하지는 않았다. 다음 단계는 기존 시군구 상권 취약도와 반경 기반 상권 취약도를 비교한 뒤 최종 산식에 반영하는 것이다.

### 위험점수 임계값

현재 위험점수 가중치와 임계값은 정책 해석형 산식이다. 이를 보완하기 위해 고립도/상권/지역감소 임계값 민감도 분석을 추가했다.

생성 산출물:

```text
outputs/reports/risk_threshold_sensitivity.csv
```

분석 범위:

```text
연도: 2029, 2035, 2040
고립도 임계값: 60, 70, 80
상권/지역감소 임계값: 60, 70, 80
```

이 리포트로 임계값을 바꿨을 때 고위험 학교 수가 얼마나 변하는지 설명할 수 있다.

## 4차 지도 UI 및 EDSS 산식 정리

### EDSS를 위험 피처 선택 체크박스에서 제외

EDSS 백분위는 폐교 확률이 아니라 과거 EDSS 학교ID 소멸 패턴과의 전국 상대순위이므로, 최종 지도에서 사용자가 임의로 켜고 끄는 체크박스에서는 제외했다. 다만 기본 위험점수 산식의 고정 보조지표로는 유지했다.

반영 내용:

```text
src/project_config.py
src/models/apply_cohort_scenario_to_risk.py
src/models/final_national_training_pipeline.py
src/viz/build_final_interactive_school_risk_map.py
```

현재 EDSS는 팝업의 `EDSS 유사도`로 표시되며, `EDSS 상위 10%`일 때 고정 가산점으로 반영된다.

### 표시단위별 범례 분리

### 교육공백 보호군 제외 통폐합 후보모델 추가

고립도가 높은 학교는 학생수가 적더라도 폐교하면 교육공백이 커질 수 있다.  
따라서 보호군을 먼저 분리한 뒤, 남은 학교 중 통폐합 가능 후보를 분류하는 정책형 보조모델을 추가했다.

신규 코드:

```text
src/models/train_consolidation_candidate_classifier.py
```

보호군 조건:

```text
5km 내 같은 학교급 대체학교 없음
또는 가장 가까운 같은 학교급이 5km 초과
또는 고립도 높고 가까운 대체학교 없음
```

결과 파일:

```text
outputs/reports/education_access_protected_summary_2026_2040.csv
outputs/reports/consolidation_candidate_classifier_metrics.csv
outputs/reports/consolidation_candidate_classifier_predictions.csv
outputs/reports/classifier_design_comparison_with_protected_filter.csv
docs/feature_variables/교육공백_보호군_후보모델_실험.md
```

현재 성능:

```text
Base Logistic F1: 0.920
Tuned HistGB F1: 1.000
```

보호군 제외 효과:

```text
2026~2040 보호군-통폐합 후보 겹침 합계: 0건
```

따라서 현재 최종 산식은 이미 고립 학교를 통폐합 가능 후보가 아니라 교육공백 우려 쪽으로 분리하고 있다. 새 모델은 후보군을 크게 바꾸기보다, 이 정책 구조를 분류모델/리포트 형태로 검증해 보여주는 역할이다.

해석 주의:

```text
이 성능은 실제 폐교 확정 라벨 예측 성능이 아니라,
현재 정책 산식으로 만든 보호군 제외 후보 label을 재현하는 성능이다.
```

## 6차 최종 위험등급 산식 Backtest

기존 EDSS 분류모델 성능은 최종 산출물 평가가 아니라 중간 보조모델 평가라는 한계가 있었다.  
이를 보완하기 위해 최종 위험등급 산식 구조를 과거 EDSS 패널에 최대한 재현해 5년 뒤 학교ID 소멸 여부로 backtest했다.

신규 코드:

```text
src/reports/evaluate_final_policy_backtest.py
```

평가 방식:

```text
2011~2015: threshold 보정
2016~2018: 5년 뒤 EDSS 학교ID 소멸 여부로 평가
```

사용 가능 피처:

```text
저학생수 여부
학생수 1년 감소
출생/출산율/순이동 기반 학령수요 감소압력
지역 상권 proxy
```

사용 불가 피처:

```text
좌표 기반 학교 고립도
대체학교 접근성
학교 반경 상권
현재 최종 EDSS 유사도 백분위
```

대표 결과:

```text
최종 산식 backtest
Precision: 0.116
Recall: 0.593
F1: 0.194
ROC-AUC: 0.860
PR-AUC: 0.087
```

비교:

```text
중간 EDSS HistGB 분류모델 F1: 0.545
최종 산식 available-feature backtest F1: 0.194
```

해석:

```text
중간 EDSS 분류모델은 proxy label을 맞히도록 학습한 모델이라 성능이 높다.
최종 산식 backtest는 최종 정책 산식 자체를 과거 데이터로 검증한 결과라 더 엄격하다.
Precision은 낮고 Recall이 높아, 최종 산식은 폐교를 정확히 찍는 모델보다 검토 후보 누락을 줄이는 우선순위 산식에 가깝다.
```

출력:

```text
outputs/reports/final_policy_backtest_metrics.csv
outputs/reports/final_policy_backtest_predictions.csv
outputs/reports/final_policy_backtest_top_capture.csv
outputs/reports/final_policy_vs_intermediate_classifier_comparison.csv
outputs/reports/final_policy_backtest_summary.md
```

### 최종 위험 피처 기반 시간검증 모델 추가
